## 1. 제품 개요

> 티켓팅을 앞둔 사람들에게 실제 환경과 유사하게 티켓팅 경험을 제공해주는
> **모의 티켓팅 시뮬레이션 웹 서비스**

- **서비스 목적** : 실제 환경과 유사하게 티켓팅을 연습할 수 있도록 시뮬레이션 환경 제공
- **서비스 타겟** : 티켓팅을 앞두고 연습해보고 싶은 사람들

## **2. 주요 기능 ( 사용자 관점 )**

1. **다가오는 티켓팅 공연 정보**
   1. **남은 시간** : 모의 티켓팅 시작까지 남은 시간
   2. **유형** : 콘서트 | 뮤지컬 | 공연장 | 극장
      - 콘서트
      - 뮤지컬
      - 공연장
      - 극장
   3. **제공사** : 인터파크 | Yes24 | 멜론티켓
   4. **일시** : 년/월/일 시간 ( 실제 공연 티켓팅 시작 시간 **2025년 10월 11일 저녁 8시** )
   5. **장소**
   6. **가격**
   7. **예상 난이도 :** 상 | 중 | 하
   8. **예매하기 버튼**
      1. **티켓팅 시작 시간**에 도달하기 전까지는 비활성화
      2. **버튼이 활성화되면 날짜 & 시간 & 회차 선택 UI/UX**
2. **이후에 다가오는 티켓팅 일정 조회 (5개까지) : 실제 다가오는 공연 + 사이트에서 예정된 모의 티켓팅 목록**

   1. **제목(타이틀)**
   2. **일시** : 년/월/일 시간 ( 실제 공연 시간 )
   3. **모의 티켓팅 시작 시간** : 년/월/일/시간
   4. **장소**
   5. **포스터**
   6. 최신순 정렬 : 공연장 규모 & 투어 규모 기준을 충족한 공연 대상

3. **현재 환경에서의 네트워크 상태 확인**
   1. 상태 : 매우 좋음 | 중간 | 매우 나쁨
   2. 속도 : 193 mbps
   3. 응답시간 : 36ms ( ping )
      - **ping : 로컬 → 티켓팅 서버 로 요청을 보내고 돌아오기까지 시간(RTT)**
      - ping : 티켓팅 서버 → 로컬
4. **모의 티켓팅 시뮬레이션**
   1. **티켓팅 예매**
   2. **접속 대기**
      1. **대기열 진행바** : 나의 대기 순서 / 전체 대기 인원
         - 현재 대기 인원
         - 예매 성공율 : ?
   3. **보안 문자 입력**
   4. **좌석 및 인원 선택**
      - 선택된 좌석
        - 구역
        - 열
        - 번호
        - 개수
      - **좌석 등급별 색상 및 가격 시스템**
        - **VIP석**: 보라색 (154,000원)
        - **R석**: 핑크색 (143,000원)
        - **S석**: 파란색 (132,000원)
        - **A석**: 초록색 (110,000원)
        - **일반석**: 회색 (99,000원)
      - **2단계 좌석 선택 프로세스**
        **1단계: ZONE 선택**
        - SVG 기반의 공연장 맵 표시
        - 10개 구역(VIP 2개, R 2개, S 2개, A 2개, 일반석 2개)
        - 각 구역은 등급별 색상으로 표시
        - 잔여 좌석 수 표시
        - 상단에 등급별 가격 안내
        **2단계: 좌석 선택**
        - 선택한 ZONE 진입 후 개별 좌석 표시 (100석)
        - 뒤로가기 버튼으로 ZONE 선택 화면으로 복귀
        - 선택한 ZONE의 등급 색상으로 좌석 표시
        - 사이드바에 선택한 좌석의 구역명과 가격 상세 표시
        - 총 금액 자동 계산
   5. **최종 피드백**
      - 각 화면(단계)별 얼마나 오래 걸렸는지 시간 측정
      - TotalTime: 처음 ~ 끝 얼마나 시간 소요
      - 전체 사용자 대비 등수
5. **실시간 채팅 기능 : 티켓팅 전/후로 소감 및 후기 공유용**
   1. 닉네임 입력창
   2. 채팅 보낸 시간
6. **티켓팅 꿀팁 제공 : 티켓팅과 관련한 꿀팁 제공**
   1. 취소표 줍는 법
   2. 성공률 높이는 법
7. **티켓팅 어워즈**
   1. 각 티켓 예매 사이트 (인터파크, Yes24, 멜론 티켓 ) 별로 최대 트래픽 유입 순위 확인
8. **티켓팅 사이트 트래픽 분석** : 인터파크, Yes24, 멜론 티켓 등 현존하는 사이트들의 실시간 트래픽을 모니터링하고 그래프로 제공 ( JS chart by amChart )
   1. 가로축 : 날짜(시간)
   2. 세로축 : 트래픽 유입량
